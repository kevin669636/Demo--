# 第一章首波射击战的节奏

目标不是重做玩法，而是把这段战斗从“稳定持续射击”优化成：

# 快速起势 → 中段抬高 → 后段快速收束

让它更符合“验证版首分钟爽点”的要求。

-----------------------------------
【本次节奏优化目标】
-----------------------------------

当前首波射击战基础已经有了：
- 玩家底部自动射击
- 噪讯怪持续来袭
- 上方进度条推进
- 有伤害数字和暴击反馈
- 有开场提示和结尾提示

但现在整体节奏偏平，合作方看到时会觉得“能打”，但未必会觉得“越来越爽”。

请这次重点优化：
1. 开场更快进入战斗
2. 怪物刷新不要太平均
3. 中段要有一次明显爽点抬升
4. 后段不要拖，要更快收束
5. 台词条不要太频繁打断战斗

-----------------------------------
【具体修改要求】
-----------------------------------

一、开场更快起势
1. 保留开场提示（如“噪讯怪来袭！准备反击！”）
2. 但提示出现后，请更快进入第一波战斗
3. 第一批怪出现时机提前，不要让玩家等待太久
4. 目标是让玩家在极短时间内就进入“已经开打”的状态

二、怪物刷新节奏改成“先少后多”
不要全程都用比较平均的刷怪频率。

请改成：
1. 开局前几秒：先来少量怪（例如 2～3 只），让玩家快速建立“我在清怪”
2. 接着几秒：怪物数量明显增加，形成第一层压迫
3. 再往后：再抬一小波怪密度，形成中段爽点

目标：
让玩家感受到节奏在往上走，而不是从头到尾都一样。

三、中段要做一次明显的爽点抬升
在进度条大约 30%～50% 之间，请安排一次明确的节奏抬高。

建议通过以下方式之一或组合实现：
1. 短时间内怪物密度更高
2. 暴击反馈更集中出现
3. 玩家输出更有“压回去”的感觉
4. 同屏敌人更多一点，但仍然可读

注意：
- 不是无限堆怪
- 是制造一个中段高点
- 让人感觉这一波战斗进入了更爽的阶段

四、70%之后快速收尾，不要拖尾
现在首波战斗后段略有“已经快赢了，但还在打”的感觉。

请改成：
1. 当进度达到约 70%～80% 后，刷怪量快速下降
2. 剩余怪物变成明显可清掉的残兵
3. 让玩家快速进入“噪讯怪已压制！”的收束提示
4. 最后的 20% 不要再拖长

目标：
让收尾更像“我把这一波压回去了”，而不是自然慢慢结束。

五、台词条频率降低，只保留最有力的内容
当前战斗中橙色台词条存在是对的，但如果过多，会打断战斗节奏。

请优化为：
1. 一波首波射击战里，只保留 1～2 条最有力的台词
2. 不要持续频繁弹出
3. 台词条要服务“噪讯怪 / 职场压迫”的主题感，不要变成持续解释层
