--------------------------------------------------
【第一波：抓人波】
--------------------------------------------------

目标：
- 快速抓住玩家
- 让玩家立刻进入“我在清怪”的状态
- 建立首分钟战斗感

怪型：
- 只放普通噪讯怪
- 不放大怪
- 不放滑移怪
- 不放关键怪

总数：
- 18只普通怪

刷新脚本：
1. 第0–3秒：6只普通怪，从上半区不同方向陆续出现
2. 第3–4秒：4只普通怪，从上半区不同方向快速补入
3. 第4–5秒：8只普通怪，从2点到11点扇区连续压入

方向要求：
- 只从上半区 / 前方扇区来，不要从玩家背后刷怪
- 不能做成360度胡乱刷

血量要求：
- 第一波普通怪偏脆
- 建议接近1发可清，或极少数需要2发
- 目标是快速建立“爆射清怪”感

同屏上限：
- 8～10只

关键目标：
- 无特殊关键目标
- 这一波的关键目标就是“快速清掉这一波唯一一批主怪群”

爆清条件：
- 清掉这一波18只普通怪
- 第一波结束后，不再继续补第一波新怪

体感目标：
- 快
- 直接
- 立刻打起来
- 不拖

【第二波：变化波 / 识别波】
--------------------------------------------------

目标：
- 告诉玩家“这段不是只会打普通怪”
- 让战斗开始有设计感
- 为第三波高潮蓄势

怪型：
- 1个大噪讯怪
- 2个滑移怪
- 8个普通怪

总数：
- 11个单位

刷新脚本：
1. 第0–1秒：4只普通怪，从上半区不同方向陆续出现
2. 第1–2秒：2只滑移怪 + 1只普通怪
3. 第2–4秒：1只大噪讯怪 + 3只普通怪

第二波关键目标：
- 大噪讯怪

第二波爆清条件：
- 打掉大噪讯怪后，剩余怪明显更容易清掉
- 这里不要做局部清场
- 可以通过以下任一种实现：
  1. 剩余普通怪血量立即降低
  2. 剩余滑移怪移动明显变慢
  3. 剩余怪短暂失衡，更容易被扫掉

滑移怪定义（必须按这个做，不要再做短距离左右偏移）：
- 滑移怪 = 之字形向斜前方逼近
- 不是原地抖动
- 不是纯横向飘
- 不是瞬移怪
- 整体方向必须是朝玩家逼近
- 但路径是左右交替、斜向前进的“之”字折线

滑移参数建议：
- 左右滑移距离 = 向前推进距离的 3～5 倍
- 每段横向滑移：90–140 px
- 每段向前推进：20–35 px
- 每只滑移怪执行 2～3 次折线
- 每次折线切换间隔：0.35～0.55 秒
- 不要无限来回

大噪讯怪要求：
- 比普通怪大 1.3～1.5 倍
- 略耐打，但不要拖节奏
- 明显看得出它是这一波重点目标

同屏上限：
- 6～8只

体感目标：
- 有变化
- 有识别重点
- 在蓄势
- 但不是高潮波

【第三波：压迫高潮波】
--------------------------------------------------

目标：
- 成为首波射击战真正的高潮
- 让玩家感到危险逼近
- 同时让玩家能很爽地压回去

怪型：
- 1个关键怪 / 核心扩音怪
- 25只普通怪
- 第三波不要再放滑移怪
- 第三波不要再放大怪
- 第三波重点是“压迫感 + 爆清感”，不是复杂技巧怪展示

总数：
- 1个关键怪 + 25只普通怪

刷新脚本：
1. 第0–1秒：关键怪 + 9只普通怪
2. 第1–2秒：8只普通怪补入
3. 第2–3秒：8只普通怪再补入

第三波普通怪要求：
- 数量明显更多
- 更密
- 更近
- 视觉上更有“压过来”的感觉
- 但血量比前两波普通怪更脆
- 建议接近1发可清
- 目标是“看起来危险，但扫起来很爽”

第三波关键目标：
- 关键怪 / 核心扩音怪

关键怪要求：
- 视觉上比普通怪更突出
- 可比普通怪稍大，或带明显边框/标记
- 不要太耐打
- 建议约等于 6～8 发命中可击杀
- 它不是Boss，只是本波核心点

第三波爆清条件：
- 打掉关键怪后，触发局部清场

局部清场要求：
1. 只做局部，不做全屏秒杀
2. 关键怪周围一定范围内（建议半径120～180px）的普通怪：
   - 立即死亡
   或
   - 瞬间残血并立刻被扫掉
3. 必须有一个肉眼明显的小范围闪光 / 崩解反馈
4. 玩家必须明确感觉：
   - 我打掉关键点了
   - 这一片噪讯被压回去了

第三波同屏上限：
- 12～14只
- 注意：总数是25只普通怪，但要分段刷，不要一下全塞满屏幕

逼近要求：
- 第三波的压迫感来自：
  - 更多
  - 更近
  - 更密
  - 更朝玩家安全区逼近
- 不要主要靠“血厚”制造高潮

第三波结束后的收束：
1. 关键怪被打掉并触发局部清场后，不再继续刷新第三波新怪
2. 剩余怪应在短时间内被快速清掉
3. 进度条快速冲满
4. 迅速进入“噪讯怪已压制！”提示
5. 不要拖长尾巴

体感目标：
- 看起来最危险
- 打起来最爽
- 爆清最明显
- 收束最快最痛快
